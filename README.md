# 🚭 Project_02_ProteinCigaretteCompany
> **팀 ProteinCigarette의 가상 기업 브랜드 웹 페이지 제작 프로젝트**

---

## 📌 Project Overview (프로젝트 개요)
본 프로젝트는 **팀 ProteinCigarette**이 기획한 가상의 기업을 주제로 한 웹 사이트 제작 프로젝트입니다.  
웹 퍼블리싱의 기본 구조를 이해하고, HTML/CSS 기반의 UI 구성 및 Spring 프레임워크를 활용한 정적 페이지 배포 경험을 쌓는 것을 목표로 진행되었습니다.

- **진행 기간**: 2024.10 ~ 2024.10
- **주요 특징**: 브랜드 콘셉트 전달, 기업/제품 소개, Google Maps API를 활용한 위치 안내 등

## 🎯 Key Goals (개발 목적)
1. **웹 퍼블리싱 숙달**: HTML/CSS를 이용한 시맨틱 마크업 및 레이아웃 설계 능력 배양
2. **UI/UX 구현**: 브랜드 콘셉트를 시각적으로 반영한 사용자 인터페이스 구성
3. **Spring 구조 이해**: Spring 프레임워크 기반의 프로젝트 구조 및 서버 환경 학습
4. **API 연동**: 외부 API(Google Maps)를 실무에 적용하는 흐름 파악

---

## ✨ Key Features (주요 기능)
* **Brand Introduction**: 가상 기업의 비전 및 브랜드 콘셉트를 담은 소개 페이지
* **Product Showcase**: 기업 제품 라인업을 확인할 수 있는 시각적 레이아웃 구현
* **Google Maps API**: API 연동을 통한 오프라인 매장 및 본사 위치 정보 제공
* **Layout Design**: 웹 퍼블리싱 기초를 반영한 정적인 웹 페이지 구성
